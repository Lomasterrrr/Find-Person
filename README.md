# FindYoutube
![alt text](https://i.imgur.com/B8zfVNm.png)

By nickname you can find a channel on YouTube, and a github account.

## Linux
- git clone https://github.com/Lomasterrrr/FindYoutube.git
- cd FindYoutube
- g++ -lcurl main.cpp
- ./a.out

## Windows
- download git
- unzip
- cd FindYoutube
- g++ main.cpp -o FindYoutube
- open exe
